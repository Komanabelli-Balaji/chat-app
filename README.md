# Chat App

A professional full-stack chat application built with React.js, TypeScript, Node.js, and Express. This project demonstrates real-time messaging, user authentication, and media upload capabilities, providing a modern and scalable communication platform.

## Features
- Real-time messaging using WebSockets
- JWT-based user authentication and session management
- RESTful APIs for user and message operations
- Cloudinary integration for media uploads
- Reusable React components and custom hooks
- Responsive UI with optimized performance
- MongoDB for persistent data storage

## Technologies Used
- React.js
- TypeScript
- Node.js
- Express
- MongoDB
- Cloudinary
- Vite

## Project Structure
- **client/**: Frontend code (React, TypeScript)
- **server/**: Backend code (Node.js, Express)

## Getting Started
1. Clone the repository
2. Install dependencies in both `client` and `server` directories
3. Set up environment variables for MongoDB and Cloudinary
4. Run the backend server
5. Start the frontend development server

## Limitations
- No group chat or multi-user conversation support
- Message read/seen status not implemented
- No push notifications for new messages
- Limited media type support (images only)
- Basic error handling and validation
